## Quais libs já estão configuradas: 

* Styled-Components
* Jest
* React-Navigation
* React-Query
* Eslint
* Prettier
* Axios
* React-Native MMKV

## Como rodar a aplicação: 
No seu terminal

IOS
```bash
# using npm
npm run ios

# OR using Yarn
yarn run ios
```
Android 

```bash
# using npm
npm run android

# OR using Yarn
yarn run android
```
